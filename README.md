## 🗨️About me

그중에서도 특히 서브컬처 장르의 게임이 저에게 게임 개발자가 되고자 하는 계기를 선물해주었습니다.

‘내가 아니면, 우리 팀이 만든 게임을 이렇게나 좋아하는 모습을 본다면 정말 행복할 것 같다’는 생각이 들었고,  
그것이 지금까지 저의 목표가 되었습니다.
